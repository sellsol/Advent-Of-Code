# Advent-Of-Code

[Advent of Code](https://adventofcode.com/) is an advent calendar of small programming puzzles for a variety of skill sets and skill levels that can be solved in any programming language you like.

I like to use these puzzles to practice and learn programming languages, so I'll be going over the same problems with different programming languages. Most of these solutions are going to be written outside the month of December, although I participated in the main timeframe some years.
