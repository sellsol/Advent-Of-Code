# Advent-Of-Code

[Advent of Code](https://adventofcode.com/) is an advent calendar of small programming puzzles for a variety of skill sets and skill levels that can be solved in any programming language you like.

I like to use these puzzles to practice and learn programming languages, so a lot of these solutions are written and submitted outside the month of December (although I participated in the main timeframe some years).

Given that I might be unfamiliar with language syntax, these solutions are not optimised. Maybe I will go over some of them later when I am more proficient.
